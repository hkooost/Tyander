# Tyander

Tyander is a tinder-like SPA for dating with different anime characters built with React + Typescript and other technologies.

## ⚠️ IMPORTANT ⚠️
**The project is at the development stage. Therefore, some functions may be missing**

## Libraries and technologies

- [Typescript](https://www.typescriptlang.org/) as a programming language
- [React](https://react.dev/) for web and native user interfaces
- [Vite](https://vite.dev/) for faster development
- [npm](https://www.npmjs.com/) for install libraries
- [SASS / SCSS](https://sass-lang.com/) for convenient writing styles
- [axios](https://axios-http.com/ru/) for easy work with REST API
- [motion](https://motion.dev/) for animations
- [nekos.best](https://nekos.best/) for images

## Usage
```
git clone https://github.com/hkooost/Tyander.git
cd Tyander
npm i
npm run dev
```
